我的作品

Flashback News

Flashback News 每30分鐘Fox News 和 CNN 熱門的新聞頭條和屏幕截圖。後端利用以Python編寫的網頁爬蟲再收集數據及儲存到數據庫。網站由 Ngnix 伺服器供應，並以 Docker 環境裏運行。

Mapbox Blog

這個旅遊博客網頁的用戶可以標記他們的旅遊位置和分享評論。前端由 Vue.js 構建 、以 Mapbox API 來繪制地圖。後端的登錄系統和實時數據庫由 Google Firebase 管理。

Youtube Music Player

以Javascript 和 SASS 建造的音樂播放器，包括音樂控制，播放列表等常用的功能。利用 Youtube Player API 及 Youtube Data API 構建音樂播放及搜索功能。
